# 🏛️ Portal de Denúncias de Lixo Urbano

Este projeto é um site simples que simula um **portal de prefeitura** onde cidadãos podem registrar denúncias de excesso de lixo nas ruas.

## 📌 Funcionalidades

- Informar a **rua do problema**
- Escrever **descrição da denúncia**
- Enviar **foto do local**
- Visualizar **prévia da imagem enviada**

## 🛠️ Tecnologias utilizadas

## ⚠️ Observação

Este projeto é apenas um **protótipo front-end** e não salva denúncias em banco de dados.
